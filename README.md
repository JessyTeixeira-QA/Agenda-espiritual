# Agenda Espiritual

Este repositório contém os arquivos de uma aplicação web e documentação de suporte para uma **Agenda Espiritual**, focada em auxiliar **Pioneiras Auxiliares** em suas atividades e relatórios.

## Conteúdo Principal

*   **Componentes da Aplicação:** Arquivos React (`.jsx`) como `Dashboard.jsx`, `Programacao.jsx`, `Relatorios.jsx`, `EditorCartas.jsx` e outros, que compõem a interface e a lógica da aplicação.
*   **Documentação de Design e Usuário:**
    *   `Documento de Design - Site para Pioneiras Auxiliares.md`
    *   `Manual do Utilizador - Pioneira Auxiliar.md`
*   **Relatórios e Resumos:** Documentos que detalham o desenvolvimento, testes e melhorias implementadas na aplicação.
*   **Tarefas Pendentes:** O arquivo `todo.md` lista as tarefas a serem realizadas.
