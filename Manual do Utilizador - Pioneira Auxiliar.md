# Manual do Utilizador - Pioneira Auxiliar

## Sobre o Site

O **Pioneira Auxiliar** é uma aplicação web desenvolvida especificamente para auxiliar pioneiras das Testemunhas de Jeová na organização das suas atividades de pregação e na criação de cartas para o ministério.

## Funcionalidades Principais

### 1. Dashboard (Página Inicial)
- **Estatísticas do Mês**: Visualize rapidamente suas horas de pregação, revisitas, estudos bíblicos e publicações do mês atual
- **Próximas Atividades**: Veja suas atividades programadas para os próximos 7 dias
- **Acesso Rápido**: Botões para criar rapidamente novas atividades e cartas

### 2. Programação
- **Gestão de Atividades**: Adicione, edite e remova atividades como reuniões, pregação, revisitas e estudos bíblicos
- **Categorização**: Organize suas atividades por categoria (reunião, pregação, revisita, estudo bíblico, assembleia, congresso, outro)
- **Detalhes Completos**: Para cada atividade, pode definir:
  - Título e descrição
  - Data e horários (início e fim)
  - Categoria
  - Horas de pregação (para atividades de pregação/revisita)
  - Número de publicações distribuídas
- **Filtros**: Filtre atividades por categoria para melhor organização
- **Estatísticas Automáticas**: O sistema calcula automaticamente suas estatísticas mensais

### 3. Editor de Cartas
- **Modelos Pré-definidos**: Três modelos prontos para usar:
  - Carta de Apresentação
  - Convite para Reunião
  - Seguimento de Visita
- **Editor Rico**: Ferramentas de formatação (negrito, itálico, sublinhado)
- **Campos Dinâmicos**: Inserção rápida de campos como [Nome] e [Data]
- **Gestão de Cartas**: Salve, carregue, edite e apague suas cartas
- **Exportação**: Imprima suas cartas ou copie o texto para outros aplicativos

### 4. Configurações
- **Informações Pessoais**: Configure seu nome, telefone, congregação e endereço do Salão do Reino
- **Assinatura Padrão**: Defina uma assinatura padrão para suas cartas
- **Backup e Restauração**: 
  - Exporte todos os seus dados para um ficheiro de backup
  - Importe dados de um backup anterior
  - Apague todos os dados se necessário
- **Estatísticas Gerais**: Veja o resumo total de suas atividades e cartas

## Como Usar

### Criando uma Nova Atividade
1. Vá para a página **Programação**
2. Clique em **Nova Atividade**
3. Preencha os campos obrigatórios (Título, Categoria, Data)
4. Adicione horários e outros detalhes conforme necessário
5. Para atividades de pregação/revisita, pode adicionar horas e número de publicações
6. Clique em **Salvar**

### Criando uma Nova Carta
1. Vá para a página **Editor de Cartas**
2. Clique em **Nova Carta**
3. Escolha um modelo pré-definido ou crie uma carta em branco
4. Use as ferramentas de formatação para personalizar o texto
5. Insira campos dinâmicos como [Nome] e [Data] conforme necessário
6. Clique em **Salvar** para guardar a carta

### Editando Atividades ou Cartas
- **Atividades**: Na página Programação, clique no ícone de edição ao lado da atividade
- **Cartas**: Na página Editor de Cartas, clique na carta desejada na lista à esquerda

### Fazendo Backup dos Dados
1. Vá para **Configurações**
2. Na secção "Backup e Restauração", clique em **Exportar Dados (Backup)**
3. O ficheiro será descarregado automaticamente
4. Guarde este ficheiro num local seguro

### Restaurando Dados
1. Vá para **Configurações**
2. Na secção "Backup e Restauração", clique em **Importar Dados**
3. Selecione o ficheiro de backup que deseja restaurar
4. Recarregue a página para ver os dados importados

## Dicas de Utilização

1. **Regularidade**: Adicione suas atividades regularmente para manter as estatísticas atualizadas
2. **Modelos de Cartas**: Personalize os modelos de cartas com suas informações pessoais nas Configurações
3. **Backup Regular**: Faça backup dos seus dados regularmente para não perder informações importantes
4. **Categorização**: Use as categorias adequadas para suas atividades para obter estatísticas mais precisas

## Armazenamento de Dados

Todos os dados são armazenados localmente no seu navegador (Local Storage). Isto significa que:
- Os dados são privados e seguros
- Não são enviados para nenhum servidor externo
- Os dados permanecem disponíveis mesmo sem ligação à internet
- É importante fazer backup regularmente, pois os dados podem ser perdidos se limpar o navegador

## Suporte

Este site foi desenvolvido especificamente para auxiliar pioneiras auxiliares das Testemunhas de Jeová. Todas as funcionalidades foram pensadas para facilitar a organização do ministério e a criação de material para pregação.

Para melhor experiência, recomenda-se o uso em computadores ou tablets, embora o site seja compatível com dispositivos móveis.
